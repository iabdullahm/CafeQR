# **App Name**: CafeQR

## Core Features:

- Customer QR Ordering Interface: Enables customers to scan a QR code, view the cafe's menu, select items, specify dine-in or car order, add to cart, and submit their order, ensuring seamless food selection and ordering.
- Cafe Admin Portal: Menu Management: Cafe owners can manage their menu items, including creating categories, adding/editing products, uploading product images, and enabling/disabling items to reflect their current offerings.
- Cafe Admin Portal: Order Management: Provides cafe administrators with a real-time view of incoming customer orders, allowing them to track and update order statuses through predefined stages (e.g., pending, confirmed, preparing, ready, completed).
- Super Admin Portal: Cafe & Basic Plan Management: Platform administrators can onboard new cafes, activate or suspend existing cafe accounts, define basic subscription plans, and assign these plans to individual cafes, enabling platform scalability.
- Secure Multi-Tenant Data Isolation: Implements a robust multi-tenant architecture with role-based access control to ensure that each cafe's data is strictly isolated and accessible only by authorized users (Super Admin, Cafe Admin, Customers).
- AI Menu Description Generator Tool: A tool integrated into the Cafe Admin Portal that leverages AI to help cafe owners quickly generate appealing and informative product descriptions for their menu items, improving efficiency in menu setup.

## Style Guidelines:

- Primary color: Deep earthy brown (#A36629). This color is inspired by coffee beans and rich cafe environments, conveying warmth and professionalism for main interactive elements.
- Background color: Soft cream (#F7F4F0). A heavily desaturated version of the primary hue, providing a clean, inviting, and comfortable backdrop for content, enhancing readability.
- Accent color: Vibrant red-orange (#E84646). Analogous to the primary but with high saturation and brightness, used sparingly for calls-to-action, alerts, and highlighting important information to draw immediate attention.
- Headline and body font: 'Inter' (sans-serif). A modern, objective, and highly readable sans-serif font suitable for all types of content across admin portals and customer interface, ensuring clarity and neutrality.
- Utilize modern, clean, and intuitive line icons for all functionalities, especially for menu categories, order status indicators, and customer navigation, ensuring universal understanding and a polished look.
- Adopt a responsive, mobile-first design strategy for the Customer Ordering Interface and a clear, dashboard-centric layout with prominent navigation sidebars for the Super Admin and Cafe Admin portals. Emphasize organized tabular data and intuitive form structures.
- Incorporate subtle and smooth animations for order status updates within the Cafe Admin Portal and provide immediate, gentle visual feedback for user interactions such as adding items to the cart or submitting orders, improving perceived responsiveness.
